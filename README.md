# 《高性能Web架构与运维实践》- 赵班长

## 前言

写这本书应该是顺势而行，因为我发现自己工作中编写的文档越来越多，越来越丰富。虽然在社区分享了很多的文档，但是发现自己写文档的时间和精力正在逐年的减少，何不将近8年的工作进行总结并分享给大家呢？我就这么干了！
虽然我并没有给这些开源软件贡献过太多的代码，但我可以将我的实践经验分享出来回报给社区，也希望越来越多的人参与进来。
之前已经编写了80%的内容，近期从Word转为markdown发布出来。

## 内容简介

  本书分为四个部分来编写的：
* 第一部分：Web架构之单机时代
* 第二部分：Web架构之集群系统
* 第三部分：Web架构之缓存应用
* 第四部分：Web架构之文件存储
* 第五部分：Web架构之MySQL扩展
